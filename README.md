# LearnMate – Course-Aligned Learning Assistant

## Overview
LearnMate is a web-based learning assistant designed to help students study using their own academic notes.  
Unlike ChatGPT or general AI tools, LearnMate answers questions strictly from user-uploaded notes, ensuring syllabus-aligned and reliable learning.

## Features
- User dashboard with learning progress tracking
- Create, view, and delete study notes
- Upload notes in PDF or TXT format
- Ask questions based only on uploaded notes
- Clean and student-friendly UI

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB Atlas
- File Upload: Multer

## How It’s Different From ChatGPT
ChatGPT answers from general internet data.  
LearnMate answers strictly from the student’s own notes, making it academically focused and course-aligned.

## Future Scope
- Teacher panel for uploading syllabus notes
- Automatic quiz generation
- AI-powered summaries from notes
